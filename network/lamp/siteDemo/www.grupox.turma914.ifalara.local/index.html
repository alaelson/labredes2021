Projeto Final de Serviços de Redes 2021

Sejam bem vindos ao Sistema Web de Demonstração de SRED

O objetivo deste sistema é ilustrar de forma simples a implementação de um servidor Web baseado em LAMP (Linux Apache PHP e Mysql)

Neste sistema podemos cadastrar, consultar e remover dados do grupos de alunos que farão o projeto final de SRED.

Desta forma, pode-se cadastrar os grupos de alunos, os integrantes de cada equipe e as máquinas virtuais (hosts) utilizadas por cada grupo.

Clique nos links abaixo:

CADASTRAR Insere informações no banco de dados.

CONSULTAR Consulta e carregar as informações armazenadas no banco.

REMOVER Remove dados armazenados no banco de dados.

ATUALIZAR Atualiza dados cadastrados no banco de dados.
